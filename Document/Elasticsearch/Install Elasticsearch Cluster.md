# Install Elasticsearch Cluster
 ```bash

Có một số mục đích chính khi tạo ra một cụm (cluster) Elasticsearch:

Tăng khả năng mở rộng và xử lý đột biến: Với cụm Elasticsearch, bạn có thể dễ dàng thêm các node mới để tăng khả năng lưu trữ và xử lý dữ liệu.  

Tăng tính sẵn sàng và dự phòng: Dữ liệu được replicate giữa các node. Khi một node gặp sự cố, các node còn lại sẽ đảm bảo hệ thống vẫn hoạt động bình thường.  

Phân tán dữ liệu và tải: Cụm cho phép lưu trữ và phân tán dữ liệu trên nhiều node, cân bằng tải trên toàn hệ thống.  

Dễ dàng quản lý và vận hành: Cụm cho phép quản lý tập trung thông qua APIs và công cụ. Các node có thể cập nhật phiên bản một cách dễ dàng.  

Hỗ trợ nhiều định dạng dữ liệu và tính năng mở rộng: Cụm ES hỗ trợ nhiều module như Graph, SQL, Vector Search, GIống như một cơ sở dữ liệu đa mục đích. 
 
Khả năng tích hợp cao với các hệ thống khác thông qua REST APIs.  

Như vậy, với nhiều ưu điểm về performance, scalability và availability, Elasticsearch cluster thường được sử dụng để xây dựng các hệ thống tìm kiếm, phân tích dữ liệu lớn.
 ```
