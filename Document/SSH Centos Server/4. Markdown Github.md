# Tổng hợp các cú pháp Markdown cho GitHub README  


### 1. Tiêu đề
```bash

# Tiêu đề cấp 1
## Tiêu đề cấp 2
### Tiêu đề cấp 3
#### Tiêu đề cấp 4  
##### Tiêu đề cấp 5
###### Tiêu đề cấp 6

```
### 2. Định dạng chữ
```bash

**In đậm**
*In nghiêng*  
~~Gạch ngang~~

```

### 3. Danh sách
```bash

- Item 1
- Item 2
  - Item 2.1
  - Item 2.2

```

### 4. Link
```bash

[Text hiển thị](https://www.example.com)
```
### 5. Hình ảnh

```bash

![Alt text](image.jpg)

```
### 6. Bảng

```bash

| Cột 1 | Cột 2 | Cột 3 |
|-|-|-|  
| Hàng 1 | Ô 2 | Ô 3|
| Hàng 2 | Ô 5 | Ô 6|

```
### 7. Block code


```bash

  ```bash
  
      // Code Here
  
  `.``

```


### 8. Gạch đầu dòng

```bash

- [ ] Task 1
- [x] Task 2

```
### 9. Thanh ngăn cách
```bash

---

```

### 10. Ký tự đặc biệt
```bash

\! \# \$ \% \&

```
### 11. Nhấn mạnh
```bash

==Nội dung cần nhấn mạnh==

```
### 12. Chú thích
```bash

Hamlet đã nói:

> To be, or not to be, that is the question

```
### 13. Emoji

```bash
:smile:

```
